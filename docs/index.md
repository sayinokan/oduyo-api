# API Dokümantasyonu

## Bu servis ile neler yapabilirim?

Ödüyo API servisi ile tanımladığınız banka hesaplarının anlık olarak bakiyelerini görüntüleyebilir, belirlediğiniz tarih aralıklarında istediğiniz bankalardan yada tüm bankalarınızdan hesap hareketlerinizi ve bu hesap hareketlerinin detaylarını çekebilirsiniz.&#x20;

Ayrıca tahsilat linkleri oluşturup müşterilerinizden ödeme alabilir , E-Ticaret altyapılarınızda Ödeme Geçidi olarak kullanabilirsiniz.

## Nereden başlamalıyım?

Öncelikle Ödüyo hesabınızda ApiKey linkine tıklayınız. Ardından mevcut API kullanıcı anahtarlarınızdan birini yada yeni oluşturacağınız anahtarlardan birini alınız.
